# Finance House Policy Chatbot POC

## Overview
An advanced RAG-based chatbot for Finance House organizational policies, featuring:
- Multi-turn conversational memory
- Query decomposition for complex questions
- Structured responses with policy citations
- Ambiguity detection and clarification
- Confidence-based escalation

## Technology Stack
- **LLMs**: Ollama (Command-R, Llama3, Phi3-mini)
- **Embeddings**: nomic-embed-text
- **Vector DB**: ChromaDB
- **UI**: Streamlit
- **Platform**: MacBook Pro M4 (24GB RAM)

## Project Structure
